# Formative-Assignment-2
 Purpose

In today's digital age, misinformation spreads faster than ever. **Life Hack or Urban Myth?** is an educational Android app designed to help users distinguish between genuine productivity tips and viral urban myths. Through an engaging flashcard quiz format, users learn valuable, safe real-world shortcuts while developing critical thinking skills.

### Why This App Matters

- **Combats misinformation**: Helps users identify common myths circulating on social media
- **Promotes safety**: Prevents users from trying dangerous or ineffective "life hacks"
- **Educational**: Provides scientifically-backed explanations for each claim
- **Engaging**: Game-based learning makes fact-checking fun and memorable

 Features

### Core Features
-  Interactive Quiz Game**: 10 carefully researched questions about common life hacks and myths
-  Instant Feedback**: Each answer comes with detailed explanations
-  Performance Tracking**: Real-time score calculation throughout the quiz
-  Modern Material Design**: Clean, intuitive interface with Material Design principles
-  Comprehensive Review Mode**: Complete review of all questions with correct answers and explanations

### User Experience
- Personalized Feedback**: Tailored messages based on final score percentage
- Smooth Navigation**: Seamless transitions between screens
- Visual Indicators**: Color-coded answers (green for correct, red for incorrect)
- Accessibility**: Large touch targets and readable text sizes

 Requirements

- Android Version**: 7.0 (API 24) or higher
- Minimum RAM**: 2GB
- Storage Space**: 50MB free space
- Internet**: Not required (offline-capable)
